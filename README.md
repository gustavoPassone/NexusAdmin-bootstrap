## 📖 Sobre
Interface administrativa responsiva desenvolvida com Bootstrap 5. O projeto simula um painel de controle (Dashboard) com visualização de métricas, gráficos e gerenciamento de pedidos.

## 🎯 Conceitos Chave
- **Sidebar Responsiva:** Uso do componente `Offcanvas` para menus móveis e posicionamento `fixed` para desktop.
- **Data Tables:** Estilização profissional de tabelas com `table-hover` e badges de status.
- **KPI Cards:** Componentes de cartão para exibição de métricas rápidas.
- **Dropdowns:** Menus interativos na navbar.

## 🛠 Tecnologias
- HTML5 & CSS3
- Bootstrap 5 (Grid, Utilities, Components)
- Bootstrap Icons
